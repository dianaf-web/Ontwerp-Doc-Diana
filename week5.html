Introductie HTML & CSS

In week 5 heb ik kennisgemaakt met Visual Studio Code. We hebben geleerd hoe een webpagina is opgebouwd met HTML. Met HTML leerden we titels, teksten, afbeeldingen en links plaatsen.

Daarnaast hebben we CSS leren gebruiken. We leerden hoe we kleuren kunnen toevoegen, ruimte tussen elementen kunnen aanpassen en hoe een pagina overzichtelijk kan worden opgemaakt.

Ook leerden we hoe we onze code kunnen opslaan en bekijken in de browser. Deze week gaf mij een basisinzicht in hoe websites worden gemaakt en hoe code en ontwerp samen werken.
